# Strict Technology Mapping & Constraints
**DO NOT DEVIATE FROM THESE VERSIONS/LIBRARIES.**

- **Framework:** Expo SDK 52 (Managed Workflow)
- **Database/Auth:** Firebase v11 (Firestore for data, Firebase Auth for users)
- **Styling:** NativeWind v4 (Tailwind CSS for React Native)
- **State:** Zustand (with Persistence)
- **Navigation:** Expo Router v4
- **Animations:** React Native Reanimated + Gesture Handler
- **Icons:** Lucide React Native
- **HTTP Client:** Axios (for Google Places API)

**Coding Rules:**
- Use Functional Components with Arrow Functions.
- Strict TypeScript: No 'any'. Every API response must have an Interface.
- Folder Structure: /components, /hooks, /services, /constants.
